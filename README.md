# Autonomous Grass Cutting Robot 

## Description
This project presents a fully functional autonomous grass cutting robot built using ESP32. The system integrates GPS-based geofencing, real-time obstacle avoidance, and intelligent path planning to ensure efficient and safe operation within a defined area.

---

## Key Features
- GPS-based boundary restriction (Geofencing)
- Real-time obstacle avoidance using ultrasonic sensors
- Intelligent path planning for systematic area coverage
- 4-wheel motor control for stable movement
- Servo motor integration for directional sensing
- Efficient and optimized embedded system design

---

## Technologies Used
- ESP32 Microcontroller
- Arduino IDE (Embedded C++)
- TinyGPS++ Library
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- Motor Drivers

---

## Future Enhancements
